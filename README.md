Posture_Detection_Using_Yolo
🎓 Student Posture Detection using YOLO

## Overview
Student engagement and classroom behavior are important indicators of learning effectiveness. This project uses YOLO 
(You Only Look Once) object detection to automatically identify and classify student postures from classroom images.
Built an AI-powered Student Posture Detection System using YOLO!
Classroom engagement is often difficult to measure at scale. To explore how Computer Vision can help, I developed a posture detection model capable of identifying multiple student behaviors from classroom images.

📌 Detected Postures:
✅ Hand Raise
✅ Look Forward
✅ Read
✅ Write
✅ Using Device
✅ Turn Head
✅ Stand
✅ Sleep

## Problem Statement

Manual classroom monitoring is time-consuming and difficult to scale. This project demonstrates how Computer Vision can 
be used to automatically analyze student posture and engagement patterns from classroom images.

## Tech Stack

* Python
* YOLO
* OpenCV
* Pandas
* Scikit-Learn
* Google Colab

## Features

* Multi-student posture detection
* Bounding box localization
* 8 posture categories
* Automatic train/validation/test split
* Data augmentation
* Performance evaluation using Precision, Recall and mAP
* Real-time inference ready

## Model Performance

* Precision: 64.41%
* Recall: 66.23%
* mAP@50: 69.03%
* mAP@50-95: 38.79%

## Key Learnings

* Handling small object detection in crowded classroom environments
* Optimizing YOLO training on limited datasets
* Improving posture classification through augmentation and hyperparameter tuning
* Building an end-to-end computer vision pipeline
